Now, I will produce a visualization using this data. This visualization will use a drop down menu to allow the user to choose which sale reigon to compare Sales in specific genres, while also using color to show which game platforms are most popular.

Top 1000 Video Game Sales in reigons compared with Genres and Color repersenting Platforms. (2017)

As stated above, this visualization allows the user to pick from multiple different reigons of sales and notice the differences of specific game genres and how they sold there. It allows the users to see the popularity of game genres in specific regions and compare that to the other reigons. With the addition of color, the user can also see the popularity of specific platforms.

For example, one could see that Role-Playing games are by far the most popular games being sold in the JP reigon, and also see that mostly Nintendo platforms are being used for these games. (3DS, DS, GBA, GB, etc.)

I used the bar mark to make this a bar graph, and also used Select for a drop down menu. That allowed me to add in the interactivity of the user being able to pick the reigon of sales. I finally added a color encoding with my x and y to make my visualization more in depth.
